# Contributing to GitDocAI Community

Thank you for your interest in the GitDocAI community! We value your feedback and participation.

## How to Participate

### 1. Report Bugs

Found a bug? Help us fix it!

1. Check if the bug has already been reported in [Issues](../../issues)
2. If not, [create a new bug report](../../issues/new?template=bug_report.yml) with:
   - Clear description of the issue
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment details
   - Screenshots if applicable

### 2. Request Features

Have an idea for a new feature?

1. Check [existing feature requests](../../discussions/categories/ideas) to see if it's been suggested
2. If not, start a new discussion in the [Ideas category](../../discussions/categories/ideas)
3. Describe:
   - What problem it solves
   - How you envision it working
   - Why it would be valuable

### 3. Join Discussions

- Ask questions in [Q&A](../../discussions/categories/q-a)
- Share your experience and use cases in [Show and Tell](../../discussions/categories/show-and-tell)
- Participate in [General discussions](../../discussions/categories/general)

### 4. Help Others

- Answer questions from other community members
- Share tips and best practices
- Point users to relevant documentation
- Help troubleshoot issues

### 5. Share Your Work

Created something cool with GitDocAI?

- Share tutorials or blog posts
- Create video guides
- Share example use cases
- Help others learn from your experience

## Community Guidelines

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md). We are committed to providing a welcoming and inclusive environment for everyone.

## Stay Updated

- Follow our [Announcements](../../discussions/categories/announcements) for new releases and updates
- Check our [Roadmap](../../projects) to see what we're working on
- Subscribe to discussions you're interested in

## Questions?

Have questions? Start a discussion in our [Q&A category](../../discussions/categories/q-a) and we'll be happy to help!

Thank you for being part of the GitDocAI community!
